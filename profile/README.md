## 🤝 Cómo Colaborar | How to Contribute

### 🇪🇸 Procedimiento para Colaborar

1. **Fork** el repositorio que te interesa
2. Crea una rama para tu feature: `git checkout -b feature/amazing-feature`
3. Commit tus cambios: `git commit -m 'Add amazing feature'`
4. Push a tu rama: `git push origin feature/amazing-feature`
5. Abre un Pull Request describiendo tus cambios
6. ¡Espera revisión y merging! 🎉

### 🇬🇧 Contribution Procedure

1. **Fork** the repository you're interested in
2. Create a feature branch: `git checkout -b feature/amazing-feature`
3. Commit your changes: `git commit -m 'Add amazing feature'`
4. Push to your branch: `git push origin feature/amazing-feature`
5. Open a Pull Request describing your changes
6. Wait for review and merging! 🎉
